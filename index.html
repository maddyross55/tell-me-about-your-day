How's your day going ?

Description

This is a generative web application. Use the interface on the right to manipulate the different parameters given! Tell me about how your day went or just type up how you're feeling right now. Play around with the colors and have some fun.

Parameters:

  • Type : Type whatever you want.
  • Tile : Change the tile color.
  • Background : Change the background color.
    Text : Change the text color.
